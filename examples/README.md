# Examples

This root owns minimal, executable native, browser, server, and SDK examples. Each example states the exact feature and maturity it proves and links to reproducible evidence.

- [`native-toolchain/`](native-toolchain/README.md) links and executes the native host boundary and emits an explicit non-database JSON report.
- [`browser-toolchain/`](browser-toolchain/README.md) validates, bundles, and instantiates the browser Wasm boundary while visibly listing unimplemented database behavior.

[`examples.json`](examples.json) is the machine authority for fixed commands, sources, outputs, and
the shared claim boundary. These examples prove toolchain paths only. New examples may not imply
document, query, storage, durability, GPU, network, compatibility, security, package, or release
functionality before the owning plan tasks and evidence exist.
