# Benchmark Report Indexes

Reserved for reviewed compact structured summaries and immutable links/hashes for retained product benchmark results. No result is accepted merely by placing a file here.

The observational workflow writes machine-specific output to ignored `dist/benchmarks/baseline/`
and `dist/benchmarks/hdoc-v1/`; CI retains both raw/summary pairs for 30 days. Task evidence may
promote a real pair with immutable hashes, but neither runner commits a result here automatically.

`host-boundary-2026-07-12.json` is the promoted P04-016 local observation. It records the dirty
pre-commit source state and exact runtime identities; it supports the transport decision only as
one correctness-checked observation, never as a universal performance threshold.
