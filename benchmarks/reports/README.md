# Benchmark Report Indexes

Reserved for reviewed compact structured summaries and immutable links/hashes for retained product benchmark results. No result is accepted merely by placing a file here.

The observational Phase 2 calibration writes machine-specific output to ignored `dist/benchmarks/baseline/`; CI retains that two-file artifact for 30 days. Task evidence may promote a real raw/summary pair with immutable hashes, but the runner never commits a result here automatically.
