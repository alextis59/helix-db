# Benchmark Datasets

This directory is reserved for substantive product benchmark dataset specifications, generators, schemas, licenses, manifests, and hashes.

The Phase 2 harness dataset is defined directly in [`harness-calibration-v1.json`](../workloads/harness-calibration-v1.json): a repository-generated MIT-licensed 1 MiB LCG32 buffer with a fixed seed, final state, and SHA-256. It is intentionally not presented as a document, query, storage, or representative product dataset.
